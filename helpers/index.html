Helper Tools

Utility pages for importing, migrating, and managing your reviews.

Import IMDb Ratings

Bulk import your IMDb ratings from a CSV export. Automatically maps IMDb scores to your 1–10 scale and sets the type to Movie or TV Show.

Import

Import BGG Collection

Bulk import your BoardGameGeek collection from a CSV export. Imports rated games with player count, year, and BGG link.

Import

Import Playnite Library

Bulk import your Playnite game library from a CSV export. Uses Completion Status and User Score columns. Skips games marked "Not Played".

Import

Cleanup Bad Imports

Delete reviews that were added via bulk import and don't have valid data. Useful for fixing botched imports without affecting manually-added reviews.

Maintenance

Migrate to Per-User Structure

One-time migration tool that moves reviews from the old flat structure (/reviews/{id}) to the per-user structure (/reviews/{uid}/{id}). Only needed if you had data before multi-user support was added.
